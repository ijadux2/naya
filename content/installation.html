Installation

Get Naya up and running on your system in minutes

Automatic Installation (Recommended)

The easiest way to install Naya with all dependencies and Neovim integration.

Clone the repository
git clone https://github.com/username/naya.git
Download the Naya source code
Run the installer
./install.sh
Linux/macOS: Use the shell script
.\install.ps1
Windows PowerShell: Use the PowerShell script
install.bat
Windows CMD: Use the batch script
Verify installation
naya --version
Check that Naya is properly installed

Installation Options

  • --dev - Development mode (symlinks instead of copying)
  • --skip-neovim - Skip Neovim configuration
  • --skip-deps - Skip dependency installation
  • --help - Show help message

Manual Installation

For advanced users who want full control over the installation process.

1. Create Installation Directory
mkdir -p ~/.local/share/naya
Create the Naya installation directory
2. Copy Files
cp naya.py naya_build.py lsp_server.py *.md ~/.local/share/naya/
Copy core files
cp -r examples syntax snippets ftdetect ~/.local/share/naya/
Copy supporting files
3. Create Command Wrappers
cat > ~/.local/bin/naya << 'EOF'\n#!/bin/bash\nNAYA_DIR="$HOME/.local/share/naya"\npython3 "$NAYA_DIR/naya.py" "$@"\nEOF
Create command line wrappers
4. Update PATH
export PATH="$PATH:$HOME/.local/bin"
Add to your shell configuration

Troubleshooting

"Command not found" Error

Cause: Naya binaries are not in your PATH

Solution:

  • Restart your terminal after installation
  • Manually add to PATH: export PATH="$PATH:$HOME/.local/bin"
  • Check if installation completed successfully

Python Not Found

Cause: Python 3.8+ is not installed or not in PATH

Solution:

  • Install Python from python.org
  • On Windows, make sure to check "Add Python to PATH"
  • Verify installation: python3 --version

Neovim LSP Not Working

Cause: LSP server not starting or configuration issues

Solution:

  • Check Neovim version: nvim --version (should be 0.5+)
  • Verify LSP server: naya-lsp --version
  • Check Neovim configuration for errors
  • Restart Neovim after installation
